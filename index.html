Deploy hexo to github

Hi, today I finally done with my first blog’s deployment.
I would like to record this in case of forgetting.

Since I am just in an entry level, so please don’t laugh

Start:

  1. Download git and node.js and hexo
  2. Deploy
    deploy: type:git repo:http://github.com/username/username.github.io.git branch:master
  3. Install npm install hexo-deployer-git
  4. ` hexo generate cp -R public/* .deploy/username/username.github.io
    cd .deploy/username/username.github.io
git add .
git commit -m “update”
git push origin master`
